1. relative 定位（保留原空间，相对自身原位置偏移）

虚线框是元素原位置，蓝色背景框是relative定位后位置（向下10px，向右20px），原空间仍保留

原位置（虚线框）
relative定位后（向下10px，向右20px）

2. absolute 定位（基于非static父容器定位）

绿色背景父容器（relative定位），红色子元素absolute定位在父容器右上角

父容器（position: relative）
child（absolute）

3. fixed 定位（固定在视口，不随滚动变化）

蓝色圆形按钮固定在浏览器视口右下角，滚动页面也不会改变位置

↑

4. sticky 定位（滚动到指定位置后固定）

向下滚动页面，导航栏会粘在顶部（top: 0），滚动回顶部则恢复原位置
